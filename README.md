# Proyecto 4

## Descripción
Este proyecto consiste en un programa compilado desarrollado en Visual Studio Code, el cual muestra un mensaje de "Hola Mundo". Es un ejemplo básico para comprender la estructura y ejecución de un programa.

## Prerrequisitos
Para ejecutar este proyecto en Ubuntu es necesario contar con:
- Sistema operativo Ubuntu
- Visual Studio Code instalado
- Compilador del lenguaje utilizado (por ejemplo, gcc para C/C++ o Node.js para JavaScript)
- Terminal

## Pasos para su ejecución

1. Actualizar el sistema:
   sudo apt update

2. Instalar Visual Studio Code:
   sudo snap install code --classic

3. Instalar el compilador (ejemplo para C/C++):
   sudo apt install build-essential

4. Abrir Visual Studio Code.

5. Abrir la carpeta del proyecto.

6. Compilar el programa desde la terminal integrada:
   (Ejemplo en C++)
   g++ programa.cpp -o programa

7. Ejecutar el programa:
   ./programa

8. Visualizar el resultado en la terminal.
